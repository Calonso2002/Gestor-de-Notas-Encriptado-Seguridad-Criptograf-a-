Este software permite crear notas de texto secretas y las encripta utilizando un algoritmo de cifrado simétrico (AES-128). Si alguien abre el archivo donde se guardan las notas, solo verá símbolos raros ilegibles. Solo se pueden leer si se introduce la contraseña correcta dentro del programa.

🛠️ Nota para VS Code: Requiere instalar una librería de criptografía. Abre tu terminal y escribe:
pip install cryptography
